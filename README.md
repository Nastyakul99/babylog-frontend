# babylog

## Описание
babylog — это трекер для учета и анализа сна, кормлений, досуга и развития навыков вашего ребенка. Помогает удобно следить за режимом и прогрессом малыша.

## Основные функции
- Ведение записей о сне, кормлениях, играх и навыках
- Анализ данных для выявления закономерностей
- Удобный интерфейс для быстрого добавления информации
